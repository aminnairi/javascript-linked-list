# @aminnairi/linked-list

## Requirements

- Node

## Installation

```console
$ npm install @aminnairi/linked-list
```

## Usage

```javascript
"use strict";

const {linkedList} = require("@aminnairi/linked-list");

const myLinkedList = linkedList(2).addFirst(1).addFirst(0);

console.log(Array.from(myLinkedList));
```
